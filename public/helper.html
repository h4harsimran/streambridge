Emby User ID and Access Token Helper

This tool helps you get your User ID and Access Token to use with the StreamBridge addon for Stremio.

To use the StreamBridge addon with your Emby server, follow these steps:

  1. Enter the URL of your Emby server (e.g., https://abcxyz.com).
  2. Enter your Emby Server username and Emby server password.
  3. Click "Get Access Info" to retrieve your User ID and Access Token.
  4. Then click the link below "Link to StreamBridge Stremio addon" and configure it with the values you just retrieved.
